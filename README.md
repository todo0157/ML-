# 머신러닝 학습 저장소

핸즈온 머신러닝 3판 학습 및 실습 자료를 저장하는 저장소입니다.

## 📚 구성

- `handson-ml3/`: 핸즈온 머신러닝 3판 실습 노트북 및 자료

## 🛠️ 환경 설정

### Python 버전
- Python 3.12.7

### 주요 라이브러리
- TensorFlow 2.19.1
- scikit-learn 1.3.2
- NumPy 1.26.4
- Pandas 2.1.4
- Matplotlib 3.8.4
- JupyterLab 4.0.13

## 🚀 실행 방법

```bash
# 가상 환경 활성화
cd handson-ml3
.\venv\Scripts\Activate.ps1

# JupyterLab 실행
jupyter lab
```
